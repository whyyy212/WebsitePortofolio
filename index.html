Tentang Saya

Wahyudi Syahputra Siregar S.Kom

Perkenalkan, Saya Wahyudi Syahputra Siregar umur 22 tahun. Visi pribadi saya adalah menjadi agen perubahan positif melalui teknologi informasi (TI). Saya percaya bahwa TI memiliki potensi besar untuk meningkatkan kehidupan manusia dan memecahkan berbagai masalah di dunia ini. Visi saya adalah menggunakan keterampilan dan pengetahuan saya dalam bidang IT untuk menciptakan solusi inovatif yang memberi dampak positif bagi Perusahaan.

Kemampuan

HTML & CSS

Membuat struktur halaman web yang semantik dan terorganisir. Mengintegrasikan berbagai elemen seperti formulir, tabel, multimedia, dan tautan dengan baik. Mengoptimalkan elemen HTML sesuai standar web untuk SEO dan aksesibilitas. Mendesain antarmuka pengguna yang responsif dengan media query. Menggunakan Flexbox dan Grid untuk tata letak halaman yang dinamis dan konsisten. Menerapkan animasi dan transisi untuk meningkatkan pengalaman pengguna. Kustomisasi desain menggunakan pseudo-class dan pseudo-elements.

FIGMA & CANVA

Kemampuan saya dalam Figma mencakup: Membuat prototipe interaktif untuk website dan aplikasi. Mengelola desain dengan sistem grid dan komponen yang terstruktur. Berkolaborasi secara real-time dengan tim desain dan pengembang. Menggunakan plug-in untuk meningkatkan efisiensi desain.

Kemampuan saya dalam Canva meliputi: Membuat desain profesional untuk media sosial, presentasi, dan materi pemasaran. Mengoptimalkan penggunaan template dan elemen grafis bawaan untuk hasil yang kreatif. Mengedit foto serta menyusun layout yang menarik secara cepat. Mampu menyesuaikan desain dengan branding dan identitas visual perusahaan.

MICROSOFT OFFICE

Menguasai berbagai aplikasi Microsoft Office yang mendukung produktivitas dan efisiensi kerja, seperti: Microsoft Word: Terampil dalam pembuatan dokumen profesional, penataan format dokumen, serta penggunaan fitur seperti mail merge dan pengolahan tabel. Microsoft Excel: Mahir dalam pengolahan data, analisis menggunakan rumus kompleks, pembuatan grafik, serta pemanfaatan fitur PivotTable dan conditional formatting. Microsoft PowerPoint: Mampu membuat presentasi yang menarik dan informatif dengan desain visual yang profesional, serta mengoptimalkan animasi dan transisi..
